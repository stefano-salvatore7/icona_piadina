# Registro delle Modifiche (Changelog) - EVO Icona Pasti

Tutte le modifiche rilevanti a questo progetto saranno documentate in questo file.

## [1.9] - 2026-01-27
### Aggiunto
- Calcolo automatico del numero totale di pasti consumati nel mese.
- Rinomina dinamica dell'intestazione della colonna da "Pasti" a **"Ticket: n"**.
- Aggiornamento della documentazione README con istruzioni dettagliate per Chrome ed Edge.

## [1.8] - 2026-01-26
### Corretto
- Ripristinato il sistema di controllo `setInterval` per garantire il caricamento dell'icona anche su connessioni lente o tabelle generate in asincrono.
- Risolto bug che impediva l'esecuzione dello script su alcuni browser Chromium-based.

## [1.6] - 2026-01-25
### Modificato
- Ottimizzazione delle dimensioni dell'icona (50x50px) per una migliore integrazione visiva.
- Miglioramento dei selettori CSS per puntare con precisione alla colonna 11 (indice 10).

## [1.3] - 2026-01-20
### Aggiunto
- Versione iniziale dello script.
- Sostituzione del valore testuale "1" con l'immagine della piadina caricata da GitHub.
- Supporto per il dominio `personale-unibo.hrgpi.it`.
